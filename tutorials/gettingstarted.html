Getting Started

  • Get the latest GGUI library and header at: GitHub.
  • Create your simple Hello World cpp file:
    • #include <GGUI.h> using namespace GGUI; int main() { GGUI::GGUI( title("Welcome to GGUI!") | node(new textField( position(STYLES::center) | text("Hello World!") )) ); while (true) {} // Keep the app running until ctr+c }
  • Link our small app with GGUI:
    • g++ main.cpp -o main -I/path/to/GGUI -L/path/to/GGUI -lGGUI
  • Optionally add GGUI to meson.build
    • Add the following lines to your meson.build file:
      • project('my_ggui_app', 'cpp') # Try to find GGUI as a system dependency first ggui_dep = dependency('GGUI', required: false) # If not found as system dependency, you might need to use find_library if not ggui_dep.found() ggui_lib = find_library('GGUI', dirs: ['/path/to/GGUI']) ggui_inc = include_directories('/path/to/GGUI') ggui_dep = declare_dependency(dependencies: ggui_lib, include_directories: ggui_inc) endif # Create the executable and link with GGUI executable('main', 'main.cpp', dependencies: ggui_dep)
  • Optionally add GGUI to CMake
    • Add the following lines to your CMakeLists.txt file:
      • # Find GGUI library find_library(GGUI_LIBRARY NAMES GGUI PATHS /path/to/GGUI REQUIRED ) # Include GGUI headers include_directories(/path/to/GGUI) # Create executable add_executable(main main.cpp) # Link with GGUI target_link_libraries(main ${GGUI_LIBRARY})

For more in-depth usage of GGUI see the Examples section.
