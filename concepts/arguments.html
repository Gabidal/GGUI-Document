Arguments - CLI parameters for GGUI

GGUI provides configurable parameters on call-time. The following supported arguments are:

Type Argument Default value Description
integer mousePressCooldown 365 The cooldown time (in milliseconds) for mouse left/right click held down to trigger for dragging or anything else.
flag enableWordWrapping true Enable word wrapping for text rendering in terminals where on row exceed wont automatically make a newline.
flag enableGammaCorrection false Enable gamma correction for colors, making them appear more vibrant.
string loggerFileName ggui.log The file name for the logger output file. Default is "GGUI.log"
flag enableDRM false Disables TTY ansi output, for DRM based pixel rendering via the GGDirect backend.
flag help/h NaN Displays the help message with all arguments and their description.

GGUI removes hyphens from the arguments, so calling --help is same as -help or help
