Lightweight terminal user interface

GGUI::SYMBOLS

Contains commonly used Unicode characters and other constant used to render the UTF elements to the finalized buffer.

GGUI::TIME

Contains useful time constants to scale with.

GGUI::BUTTON_STATE

Used to store the keyboard states for every individual state.

GGUI::CONSTANTS

Contains constants about ASCII Escape Code Sequences.

GGUI::BUTTON_STATES

Contains all the key values for the KEYBOARD_STATES

GGUI::RGB

The main class to represent colors in abstract format, before it gets transformed into UTF.

GGUI::RGBA

Extends: RGB class, to add Alpha channel.
